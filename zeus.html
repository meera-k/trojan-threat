Zeus

This very dangerous trojan attack was first detected in 2009, when it was found to be stealing information from the US Department of Transportation. The original Zeus trojan, developed in 2007, worked by taking the form of spam emails, online advertisements, and false links pushed by search engines through a process known as SEO* poisoning.

* SEO stands for Search Engine Optimisation, and it is essentially a means of gaming the system by which search engines index websites and rank them. By employing SEO tactics, you can make sure that your website will rank higher in the index and thus be more likely to be clicked on by someone who has just made a query (ie. searched for something online).

Where did it come from?

27-year-old computer science graduate Hamza Bendelladj developed the Zeus trojan attack in 2007 to steal money from US agencies (eg. banks, financial establishments, etc.). The millions of dollars he stole were given to Palestinian charities. For this reason, Bendelladj is considered by some to be a hero, and by others to be a villain.

What were the consequences?

In July 2009, after it was first detected, security software company Prevx reported that the Zeus trojan had compromised almost 90,000 FTP* credentials from prominent corporate websites, including sites like Bank of America, Amazon, Cisco, and NASA, just to name a few. In doing so, the Zeus trojan allowed for the access of a lot of sensitive information, and led to the loss of a lot of money.

* FTP stands for File Transfer Protocol, and it is exactly what it sounds like.

How exactly did it work?

Firstly, it had to infect the user’s computer. It did so by utilizing spam messages and tricking users into clicking on a link that would take them to a page on a malicious website, and by infecting downloadable files from the Internet. The Zeus trojan was able to infect websites and insert their own malicious code into downloadable files so that when a user downloads one of such files, they will inadvertently download the trojan’s harmful code as well.

From there, it joined the user’s computer to a botnet. Through keylogging and website monitoring, the trojan was able to know when the user entered sensitive information and save it on its end so that it could be used to steal the user’s money, for example.

That was then. What about now?

The source code for the original Zeus trojan was leaked in 2011 so it became useless, but since then, many versions of it have come up, all with the same main tactics and goals. Zeus Sphinx, one such variant, has surfaced recently in part due to the COVID-19 pandemic. Using government relief payments as its way in, Sphinx has been targeting banks and financial institutions in the US, Canada, and Australia. As technology advances, so do these Zeus variants, so protecting our devices is still as important as ever.

How can we defend ourselves?

Now, with different versions of the original Zeus trojan circling the globe, it has become more and more difficult to prevent these attacks. The best way to keep yourself safe is to be cautious when receiving emails and downloading files. Before clicking on any links in an email, make sure that the sender is who they claim to be, and check the content of the email. Generally, if it’s a fake email from someone you know, the message will have grammatical errors or won’t seem like something the real sender would say. Additionally, when downloading files, check to make sure that the connection on the website you are downloading from is secure. If it is, then this means that the Certificate Authority has verified that the website you are on is what it claims to be and not a fake replica designed to trick people. Overall, taking caution when you see something that seems a bit unsafe is the best way to protect yourself.
